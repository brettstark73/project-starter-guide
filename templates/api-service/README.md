## Planned Stack

### Option A: Node.js
- **Framework:** Fastify or Hono
- **Language:** TypeScript
- **Database:** PostgreSQL
- **Validation:** Zod
- **Documentation:** OpenAPI/Swagger
- **Testing:** Vitest
- **Deployment:** Docker

### Option B: Python
- **Framework:** FastAPI
- **Database:** PostgreSQL
- **Validation:** Pydantic
- **Documentation:** Auto-generated
- **Testing:** pytest
- **Deployment:** Docker
